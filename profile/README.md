# vanityURLs 

Welcome to our GitHub page! 

vanityURLs is a simple solution to deploy a budget-friendly Url Shortener. It runs at scale in your Cloudflare subscription with continuous integration.

Want to use it? Clone the [repository](https://github.com/vanityURLs/vanityURLs) or read the [installation](https://github.com/vanityURLs/vanityURLs/blob/main/doc/installation.md).
